rptools.rpfba.rpFBA.rp_fraction¶

rptools.rpfba.rpFBA.rp_fraction(rpsbml: rptools.rplibs.rpSBML.rpSBML, objective_rxn_id: str, biomass_rxn_id: str, hidden_species: List[str], fraction_coeff: float = 0.75, logger: logging.Logger = <Logger rptools.rpfba.rpFBA (WARNING)>) → cobra.core.solution.Solution¶

Optimise for a target reaction while fixing a source reaction to the fraction of its optimum

Parameters

  • source_reaction (str) – The id of the source reaction

  • source_coefficient (float) – The source coefficient associated with the source reaction id

  • target_reaction (str) – The id of the target reaction

  • target_coefficient (float) – The source coefficient associated with the target reaction id

  • fraction_of_optimum (float) – Between 0.0 and 1.0 determining the fraction of optimum (Default: 0.75)

  • is_max (bool) – Maximise or minimise the objective (Default: True)

  • pathway_id (str) – The id of the heterologous pathway (Default: rp_pathway)

  • objective_id (str) – Overwrite the default id (Default: None)

Returns

Tuple with the results of the FBA and boolean indicating the success or failure of the function

Return type

tuple
