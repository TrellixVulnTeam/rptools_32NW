rptools.rpfba.medium.df_to_medium¶

rptools.rpfba.medium.df_to_medium(df: pandas.core.frame.DataFrame) → Dict[str, float]¶

Extract a dictionary from a dataframe, fitting with cobrapy needs.

Parameters

df (pd.DataFrame) – a dataframe with ‘reaction_name’, ‘upper_bound’, ‘rp_compound’, ‘libsbml_reaction’ columns

Returns

a dictionary with keys supporting ‘reaction_name’ and values supporting ‘upper_bounds’

Return type

Dict[str, float]
