rptools.rpfba.rpFBA.write_fluxes_to_reactions¶

rptools.rpfba.rpFBA.write_fluxes_to_reactions(fluxes: pandas.core.series.Series, rpsbml: rptools.rplibs.rpSBML.rpSBML, pathway_id: str, sim_type: str, logger: logging.Logger = <Logger rptools.rpfba.rpFBA (WARNING)>) → None¶

Write Cobra results to the reactions of pathway with id pathway_id in rpsbml object.

Parameters

  • fluxes (np_series) – Fluxes.

  • rpsbml (rpSBML) – rpSBML object of which reactions will be updated with results

  • pathway_id (str) – The id of the pathway within reactions will be updated

  • objective_id (str) – The id of the objective to optimise
