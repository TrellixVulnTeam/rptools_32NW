rptools.rpextractsink.rpextractsink.write(outFile, elts, delimiter=',', quotechar='"')

Write elements of elts list into file as ‘csv’ would do

Parameters

  • file – file to write into

  • elts – list of elements to write

  • delimiter – character to insert between each element

  • quotechar – character to put around each element
