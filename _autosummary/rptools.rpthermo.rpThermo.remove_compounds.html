rptools.rpthermo.rpThermo.remove_compounds¶

rptools.rpthermo.rpThermo.remove_compounds(reactions: List[chemlite.Reaction.Reaction], rxn_target_id: str, compounds: List = [], logger: logging.Logger = <Logger rptools.rpthermo.rpThermo (WARNING)>) → Dict¶

Try to remove compounds that are unknown in the eQuilibrator cache from reaction set (pathway). For this purpose, a stoichiometric matrix is built with only coefficients of these compounds. Then, try to solve as a linear equations system and apply new coeffs to reactions
